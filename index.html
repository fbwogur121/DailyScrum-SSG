✨ Jekyll 자동화!

  1. 📄 template.md 복사해서 스크럼 작성
  2. 📁 _posts 폴더에 저장: YYYY-MM-DD-이름-제목.md
  3. 📝 Front Matter에 작성자, 날짜 등 설정
  4. 🐱‍👤 GitHub에 커밋 & 푸시
  5. 🚀 GitHub Pages가 자동으로 빌드!

📋 파일명 규칙
